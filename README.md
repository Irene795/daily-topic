# 高研新知每日选题系统

每天自动抓取微博热搜，筛选适合"高研新知"栏目的选题。

## 功能特点

- 每天北京时间 10:00 自动运行
- 智能筛选：排除娱乐八卦、政治敏感、已写过选题
- 备选选题库：心理学概念、社会现象、职业成长（每天轮换）
- 结果自动保存到 `reports/` 目录

## 文件结构

```
daily-topic.js       # 主脚本
package.json         # Node.js 配置
.github/workflows/   # GitHub Actions 配置
reports/             # 每日选题报告
```

## 报告内容

- 今日推荐选题（结合热搜与新知价值）
- 心理学/认知科学选题
- 社会现象/家庭选题
- 有解读价值的新闻
- 备选选题库（每天轮换）

## 已写过选题

- 同理心/共情、情感账户、心理投射、乐观偏差
- 破窗效应、鸟笼效应、巴纳姆效应、损失厌恶、框架效应
- 独居经济、孤独经济、数字游民、财务斩杀线
- 耐心资本、信息差、正念、情价比
- 选择能力、沟通境界、输出力、张雪峰事件
